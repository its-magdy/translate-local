# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

> **Note:** This root changelog tracks project-wide milestones. Individual packages under `packages/` maintain their own changelogs and version independently (see each package's `CHANGELOG.md`).

## [0.1.0] - 2026-03-30

### Added

- CLI-first translation tool with TranslateGemma via Ollama
- Glossary enforcement with CSV import/export and domain filtering
- Context-aware translation using TF-IDF retrieval
- Interactive terminal UI (TUI) with tab navigation
- Pluggable model adapter architecture
- Image translation support
- Streaming output support
- JSON output mode
- JSONC configuration at ~/.config/tl/config.jsonc
